기술블로그를 작성하는데 있어서 가장 필수 적인 기능이 아닐까 싶다.

## how to use

> \`\`\` [language]  
> code source  
> \`\`\`

## example

````
```js
console.log("hello world");
```
````

```javascript
console.log("hello world");
```
